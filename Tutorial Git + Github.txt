#1: Abrir a pasta que você quer colocar no Github 
#2: Botão direito -> Git Bash Here
#3: git init -> enter
#4: git add . -> enter
#5: git add remote origin [url do repositório; abrir no navegador, ir em Code -> copiar HTTPS] (nota: ESCREVA MANUALMENTE, COLAR VAI DAR ERRO!)
#6 git push origin [nome-da-branch] -> enter (pode usar [nome-da-branch] --force para criar uma branch nova se ela não existir)
#7: se der erro copia e cola a ultima mensagem do console no google
